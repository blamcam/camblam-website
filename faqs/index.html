How do I add LurkBait to my OBS?

Once LurkBait is open, it can be added to OBS via a Game Capture source with Allow Transparency turned on (recommended) or a Window Capture source with Windows 10 capture method. See the video below for a tutorial on how to do both these options in OBS.

There's a black box around the LurkBait window, I can't seem to get transparency working.

Make sure you're using a Game Capture source with Allow Transparency turned on in the source settings (recommended) or a Window Capture source with Windows 10 capture method set in the source settings. There's a video tutorial on this page on how to do this.

What can players do with gold?

Right now gold is purely a status symbol and defines the peckin' order on the leaderboards, but in a future update we intend to add a way for fishers to redeem their gold for cosmetics such as rods, lures, and bait. Keep an eye out for me details about this in the future!

Can I use a different account to post messages to chat?

In the newest version, yes you can! Log in your secondary account from the LurkBait Settings menu.

Where's the link to the overlay?

LurkBait is a desktop game, that you add to OBS as a capture source like you would any other game — however LurkBait is optimised with transparency, autohiding, and minimal performance impact to allow you to to keep LurkBait open while you play other games. Think of LurkBait as akin to Marbles on Stream and Placid Plastic Duck Simulator in the idea that it's a game that's open while your stream is running.

If you're not sure how LurkBait works or you want to test the performance of LurkBait while playing other games, feel free the download the free demo from Steam.

Do I need to launch LurkBait through Steam?

If for whatever reason you're having problems launching LurkBait through Steam — you can launch it manually by going to LurkBait in your library, clicking the ⚙️ icon, the clicking "Manage" then "Browse local files" and launching the LurkBait exe from there.

This is a great option if you have a streaming PC thats different from your gaming PC and you can't launch steam games from two computers at once.

Can I use LurkBait with a dual PC setup?

If you play games on a seperate PC from the one you stream on, you might encounter Steam preventing you from opening LurkBait — however you can launch it manually by going to LurkBait in your library, clicking the ⚙️ icon, the clicking "Manage" then "Browse local files" and launching the LurkBait exe from there.

LurkBait won't go away!

The Autohide button in the lower left corner of the window will turn on Autohide, which will transition the window in and out when needed.

LurkBait is chugging like crazy!

If you're having performance problems with Lurkbait, try lowering the resolution in the LurkBait Settings and restarting the app, turning off High details and lowering the graphics quality settings.

LurkBait is wayyyy too big for my screen!

You can shrink LurkBait by lowering the resolution in the LurkBait Settings and restarting the app.

Are custom catches included in the normal catch pool?

If Custom only mode is DISABLED in the LurkBait Settings panel (the default setting), custom catches will be included in amongst the regular catch pool. If Custom only mode is ENABLED, only the custom catches will be avaiable for players to catch (as long as you have 1 or more custom catches), no in-built catches will show up.

Is there a limit to the number of custom catches I can add?

Nope! People have added hundreds without impacting performance.

What size of image do you recommend for custom catches?

It's not a rule but I'd recommend a square transparent PNG or GIF with a maximum size around 1000x1000 pixels. Images that are too large may slow down LurkBait.

Can I use LurkBait if I'm an IRL streamer?

If you use any other application to stream other than OBS, you unfortunately cannot use LurkBait. LurkBait currently does not work with streaming from the Twitch Mobile App, or directly from a GoPro.

Can I customize the message that LurkBait posts to chat when something is caught?

Yes you can! If you enable Send catch results to chat and Custom chat results message you can input a custom response message and use the following variables to format the message:

{user}

Tags the user who caught it

{rarity}

The rarity of the catch

{rarityarticle}

The word that comes before the rarity name eg. "a" or "an"

{catch}

The name of the catch

{catcharticle}

The word that comes before the catch name eg. "a" or "an"

{description}

The catch description

{weight}

The catch weight

{gold}

The catch gold value

{rating}

The catch rating as a number

{ratingstars}

The catch rating as 1-3 ⭐️ emojis

{playergold}

The players gold based on current leaderboard setting

{playergoldalltime}

The players all time gold

{playergoldleaderboard}

The players gold in the current leaderboard

For example:

{user} you caught {rarityarticle} {rarity} {ratingstars} {catch} weighing {weight} worth {gold} gold! You now have {playergold} gold!

Will output:

@CamBamThanksMan you caught a Rare ⭐⭐⭐ Rubber Ducky weighing 0.12kg worth 120 gold! You now have 432 gold!

LurkBait missed someones cast! And they paid for it too...

If for whatever reason LurkBait misses someones chance to fish, the broadcaster and mods (if the Allow mods to target cast option is enabled) can use !fish @username to trigger the cast manually. You as the broadcaster can also give a chatter multiple casts using !fish @username x5 to trigger 5 casts for example.

What other chat commands can I use?

Here are some other inbuild chat commands you can use.

The broadcaster account and the chatbot account are always superusers. Mods can be superusers be enabling the Mods are superusers option.

!fish @username

Queues up cast for username. Broadcaster only unless mods enabled

!fish @username x5

Queues up x amount of casts casts for username, max 20. Superuser only

!deletefisher @username

Deletes username from records, will ask to confirm. Superuser only

!resetleaderboard

Starts a new leaderboard. Superuser only

!fishboost

!fishboost on

!fishboost off

Turns on or off "Boost chances of ultra rare fish". Superuser only

!switchleaderboard

!switchleaderboard alltime

!switchleaderboard current

Switches leaderboard between all-time and current leaderboard. Superuser only

!mute

!mute on

!mute off

Toggles mute. Superuser only

!reconnect

Attempts to reconnect. This only works if chat is still connected but Twitch events have stopped working like point redeems, bits, or subs. Superuser and moderator only

Do "Test Catches" count towards the leaderboard or the dex?

Nope! Not anymore anyway, using the Test Cast button in the bottom left of the UI will not contribute to the leaderboard, streamer gold total, or the dex progress.

How does the rarity and catch chance work for custom catches?

When adding custom catches, the following gold brackets specify the catch rarity and estimated catch chance (weight does not affect value or rarity). Junk will be added a category for custom catches soon. Big thankyou to Bydand for helping put this together.

Min Max Est. chance*With custom only mode turned OFF
Common 1 34 2.12%
Uncommon 35 59 1.06%
Rare 60 109 0.79%
Epic 110 200 0.32%
Legendary 201 1000 0.16%

Steam achievements aren't unlocking for me!

Are you running LurkBait while Steam in offline mode? If yes, just start LurkBait while Steam is online once in a while and all your achievements will unlock when LurkBait is booted. If Steam still isn't unlocking any achievements, some users reported that temporarily turning off Family Sharing will fix the achievements unlocking.
